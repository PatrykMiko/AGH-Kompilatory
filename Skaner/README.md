# Zbiór tokenów
| Nazwa/Kod | Opis                                                     | Przykładowa wartość |
|:----------|:---------------------------------------------------------| :--- |
| `INT`     | Liczba całkowita (ciąg cyfr)                             | `2`, `76`, `9` |
| `ID`      | Identyfikator (ciąg liter i cyfr, zaczyna się od litery) | `x`, `zmienna1` |
| `PLUS`    | Operator dodawania                                       | `+` |
| `MINUS`   | Operator odejmowania                                     | `-` |
| `MUL`     | Operator mnożenia                                        | `*` |
| `DIV`     | Operator dzielenia                                       | `/` |
| `LPAREN`  | Nawias otwierający                                       | `(` |
| `RPAREN`  | Nawias zamykający                                        | `)` |
| `END`     | Koniec wyrażenia                                         | *brak* |
| `ERROR`   | Token oznaczający nierozpoznany znak                     | `@`, `$` |
